## Alpha-Beta Pruning
- **How it works:** Optimizes Minimax by pruning branches that won’t affect decision outcome (α and β bounds).
- **Applications:** Chess, tic-tac-toe, other adversarial games.
- **Complexity:**
- Time: O(b^(m/2)) with effective pruning.
