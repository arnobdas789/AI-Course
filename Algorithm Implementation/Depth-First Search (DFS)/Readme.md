## Depth-First Search (DFS)
- **How it works:** Dives into the graph recursively or via a stack (LIFO), exploring one branch fully before backtracking.
- **Applications:** Cycle detection, topological sorting, maze solving.
- **Complexity:**
- Time: O(V + E)
- Space: O(V)
